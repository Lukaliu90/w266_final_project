# How to compile

## Pre-req
Install tex

## Compile
If you changed citations, run this
```
pdflatex final_paper
bibtex final_paper
pdflatex final_paper
pdflatex final_paper
```

If you changed any references, run this
```
pdflatex final_paper
pdflatex final_paper
```

If you just change text, run this
```
pdflatex final_paper
```
